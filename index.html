Abonner på hockeykamper i din kalender

Slik legger du til i kalenderen

iPhone / iPad: Innstillinger → Kalender → Kontoer → Legg til konto → Annet → Legg til abonnert kalender → lim inn URL

Android / Google Kalender: Åpne calendar.google.com → Andre kalendere (+) → Fra URL → lim inn URL

Mac (Apple Kalender): Fil → Nytt kalenderabonnement → lim inn URL

Outlook: Legg til kalender → Abonner fra Internett → lim inn URL
